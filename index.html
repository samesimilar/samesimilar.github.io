AI2CG

 

AI2CG is a free plugin I created for AI CS6 and later (currently has been tested on AI 2014). It will export CoreGraphics code for iOS (& OSX) from an Adobe Illustrator document.

 

MirrorXMLLib

 

MirrorXMLLib is an Objective-C wrapper for the libxml2 SAX XML & HTML4 parser. It lets you bind handler blocks to detected nodes using XPath-like selectors that are matched during SAX parsing. You can grab it on Github here.
